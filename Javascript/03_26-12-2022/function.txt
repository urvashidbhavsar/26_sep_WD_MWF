functions -> 
there is two type function -> (1) built-in (predefine) (2) user-define

(1) built-in -> for ex, log(), write(), writeln(), alert(), isNaN(), toggle(), add(), remove() etc..

(2) user define -> 

syntax, 
function function_name(parameters1, parameters2){

}

valid name -> func(), my_func(), my_func1(), my_func_1()

invalid name -> my func(), 1my_fun(), @my_func()

events -> 
some task perform on webpage with click, load, submit etc..

events -> 4 type of events
(1) keyboard (2) mouse (3) form (4) window

event handler -> which is handle the event.
for, click -> onclick
    submit -> onsubmit
    load -> onload
