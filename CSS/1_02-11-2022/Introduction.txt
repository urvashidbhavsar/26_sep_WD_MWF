CSS -> cascading stylesheet

why used -> to decorate out website / designing in website.

how many types of prepared CSS?
3 ways
(1) inline - apply with style attribute
(2) internal - apply with style tag in head section
(3) external - create an another file with .css extension and link with <link> tag.


CSS selector ->
(1) element selector -> in <p> tag 'p' is an element.
(2) ID selector -> apply with #. for ex, #myid
(3) class selector -> apply with '.' for ex. .myclass
(4) group selector -> h1,h2,h3
(5) universal selector -> apply on whole page (*)


inline CSS syntax :
property : value;

interna/external CSS syntax :
selector {property : value;}
